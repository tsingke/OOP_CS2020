## 一、实验目的

*  掌握类的设计方法
*  掌握visual studio 代码调试方法



## 二、实验任务

请采用C++语言设计类,并实现类. 要求程序代码:清晰简洁易读, 良好的开闭性, 编程风格规范统一。

【编程题】请设计一个栈类Stack, 通过该类实现与栈相关的各种操作功能. 

* 栈内数据初始化


* 数据进栈

* 数据出栈

* 计算当前栈的大小

* 判断栈是空的还是满的


提示:栈内数据存储时可以借助数组实现.
